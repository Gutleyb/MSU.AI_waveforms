Датасет импульсов ТАЭ, содержащий файлы волновых форм импульсов (.npz) и метаданные (.txt), размещен на платформе Kaggle.

Здесь размещен код Google Colab, в котором реализовано обучение и тестирование моделей MLP и CNN.
