В данном репозитории хранятся данные для проекта по курсу *"Нейронные сети и их применение в научных исследованиях"* от **MSU.AI**.

1. Проект посвящен анализу волновых форм сигнала с целью эффективной бинарной классификации импульсов термически стимулированной акустической эмиссии (ТАЭ): шум и полезный сигнал.

2. Датасет импульсов ТАЭ, содержащий файлы волновых форм импульсов (.npz) и метаданные (.txt), размещен на платформе **Kaggle**. Примеры файлов из датасета для одного из экспериментов (гипс) находятся в папке `DATA`. Файл `wf_example.txt` - пример волновой формы одного из сигналов, содержащихся в npz-файле.

3. Здесь размещен код *Google Colab*, в котором реализовано обучение и тестирование моделей MLP и CNN - `Waveforms_NN.ipynb`. Датасет с **Kaggle** напрямую загружается в блокнот.
